laravel-https

发表于 2017-05-24

laravel强制使用https

  1. 使用middleware

├── app
│ ├── Http
│ │ ├── Middleware

增加 app/Http/Middleware/HttpsProtocol.php

public function handle($request, Closure $next) {
    if($_SERVER['HTTP_X_FORWARED_PROTO'] == 'https') {
        \URL::forceScheme('https');
    }
}
  1. 在provider里面实现

├── app
│ ├── Providers

修改 AppServiceProvider, boot 增加：
if($_SERVER['HTTP_X_FORWARED_PROTO'] == 'https') {
    \URL::forceScheme('https');
}
